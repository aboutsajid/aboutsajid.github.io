Why trust Cipher

The distribution model is direct GitHub releases with transparent versioning, so users can verify what they install.

Local-first by default

Primary data lives on your machine. Network requests are feature-specific, explicit, and optional where applicable.

No forced account model

Tools are downloadable executables. There is no signup wall, billing gate, or hidden session requirement.

Release transparency

Version tags, binaries, and notes are publicly visible on GitHub releases for each project repository.

Simple install paths

Each app ships as a straightforward Windows download, with install steps documented directly on this page.

Install guides

Quick setup instructions for each tool. Expand the one you need.

Cipher Vault setup
  1. Download the latest installer from the Vault release button above.
  2. Install and open the app, then create a new vault with a strong master password.
  3. For browser save, open the Browser feature and load unpacked extension from the app output folder.
Cipher Screen Recorder setup
  1. Download the latest Screen Recorder executable from release assets.
  2. Launch and allow microphone/system audio access if prompted by Windows.
  3. Pick source, confirm output folder, start recording, then export your capture.
Cipher YT Downloader setup
  1. Download the latest YT Downloader setup executable from the release page.
  2. Install and open the tool, then paste video or playlist URLs.
  3. Choose format and quality, then start download or audio-only conversion.
Cipher Compress setup
  1. Open the Cipher Compress release page and download the latest setup file.
  2. Install the app, then select input files or folder and choose output folder.
  3. Pick a compression profile, run queue, and use result report to verify savings.
Cipher Ai setup
  1. Download the latest Cipher Ai setup executable from the release page.
  2. Install and launch, then open Settings to configure OpenRouter key or Ollama local mode.
  3. Choose your model in the top selector and start a new chat workflow.
