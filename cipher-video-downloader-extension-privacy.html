Privacy Policy

Cipher Video Downloader for YouTube

Last updated: April 4, 2026

This privacy policy explains how the Cipher Video Downloader for YouTube Chrome extension handles data.

What the extension does

The extension helps a user send the current YouTube video link from Chrome to the installed Cipher Video Downloader desktop app. It also lets the user save a preferred download profile and keep a small local queue inside the browser.

Data the extension accesses

  • The current YouTube page URL, so the extension can identify the video the user wants to send.
  • A locally saved default profile choice.
  • A locally saved queue of links the user chooses to keep in the extension.

How data is used

  • The current YouTube URL is used only to prepare a handoff to the installed Cipher Video Downloader desktop app.
  • The saved profile and queue are used only to support the extension's user-facing features.
  • Data is processed only for the extension's single purpose: sending YouTube links to the desktop app and supporting that workflow.

Storage

The extension stores the selected default profile and queue items locally using Chrome extension storage. This data stays on the user's device unless the user separately enables Chrome sync behavior on their own browser profile.

Data sharing

The extension does not sell user data. It does not transfer user data to advertisers, data brokers, or unrelated third parties. When the user chooses to launch the desktop app, the selected YouTube link is passed to the installed Cipher Video Downloader desktop app through the local cipheryt:// protocol workflow.

Remote code and tracking

The extension does not use remote code. It does not include analytics, advertising trackers, or hidden telemetry.

Website access

The extension runs only on supported YouTube pages needed for its handoff features, including watch pages, Shorts, live pages, and supported mobile/music YouTube pages defined in the extension manifest.

Your choices

  • You can remove saved queue items and profile settings from the extension at any time.
  • You can uninstall the extension through Chrome at any time.
